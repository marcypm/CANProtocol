Readme
=======
Folders:
	ccode: contains all the c code
		src: contains the source code
			sha256.c is the source code for the software impl as well as the legup implementation
			Makefile is the makefile used to synthesize designs by legup
		
	reports: contains all the sysntesis reports and simulations by quartus

	tex: contains all the written work

	verilog: contains all the verilog files for the hardware implementation
		Makefile is the makefile to compile and simulate the project. It uses the same targets as Legup
		sha256.v is the top level design
